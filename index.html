INTRODUCTION

Embarking on the Web Development Adventure

As a student diving into the world of web development, I'm embarking on an exciting journey to learn how to create my own website. With a passion for technology and a desire to build something from scratch, I'm eager to explore the fundamentals of HTML, CSS, and JavaScript. This website will serve as a canvas for me to experiment, learn, and showcase my progress as I navigate through the intricacies of web design and development.

Setting the Foundation: Learning HTML

My first step in this journey is to master HTML, the backbone of any website. I'm dedicating approximately 1.5 weeks to understanding the structure and semantics of HTML, learning how to create headings, paragraphs, lists, and links.

Styling the Web: Exploring CSS

After grasping the basics of HTML, I'll dive into CSS to bring my website to life with colors, layouts, and responsive design.

Adding Interactivity: Learning JavaScript

Finally, I'll venture into the world of JavaScript to add interactivity and dynamic features to my website. With another 2 weeks dedicated to learning this powerful programming language.
